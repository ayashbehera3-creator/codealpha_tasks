# CodeAlpha Cyber Security Internship

## Overview

This repository contains the projects completed as part of my **CodeAlpha Cyber Security Internship**. During this internship, I worked on practical cybersecurity tasks to strengthen my understanding of network security, phishing awareness, and secure software development.

## Completed Tasks

### Task 1 – Basic Network Sniffer

**Objective:**
Developed a Python-based network packet sniffer to capture and analyze network traffic.

**Key Features**

* Captures live network packets
* Displays source and destination IP addresses
* Identifies network protocols
* Displays packet information for analysis
* Built using Python and Scapy

---

### Task 2 – Phishing Awareness Training

**Objective:**
Created a cybersecurity awareness presentation explaining phishing attacks and how to prevent them.

**Topics Covered**

* What is phishing?
* Types of phishing attacks
* Social engineering techniques
* Identifying fake emails and websites
* Best practices for staying safe online
* Real-world phishing examples

---

### Task 3 – Secure Coding Review

**Objective:**
Performed a security review of application source code to identify vulnerabilities and recommend secure coding practices.

**Focus Areas**

* Input validation
* SQL Injection prevention
* Cross-Site Scripting (XSS)
* Authentication and authorization
* Password security
* Error handling
* Secure coding recommendations
